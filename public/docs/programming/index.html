    Variables #

    We must declare the variable type. Common data types are: string, int, float, bool.

    For data structures, the type is something like Vector3 or Quaternion.

    For classes, we use the class name.

    Declaring data variables.

    public string weUseCamelCase;//Variable names uppercase every word except the first, and cannot start with numbers. We do not use snake_case.
private int someNumber = 4;//you set their values during declaration.
